# 🤖 AI-Powered Multi-Agent System for Smart Catering Operations

## 📝 Project Overview
This project is an automated system designed to streamline catering business operations. It uses a **Multi-Agent Architecture** where specialized AI agents work together to handle the entire workflow from customer inquiry to final pricing.

## ✨ Features
*   **Customer Interaction Agent:** Captures event details and requirements.
*   **Menu Planning Agent:** Generates suitable menus based on budget and guests.
*   **Inventory & Procurement Agent:** Calculates ingredients needed and checks stock.
*   **Logistics Planning Agent:** Schedules preparation and delivery.
*   **Pricing & Optimization Agent:** Computes total cost and pricing.
*   **Monitoring Agent:** Tracks progress and ensures smooth execution.

## 🛠️ Technologies Used
*   **Language:** Python 3
*   **Concepts:** Object-Oriented Programming (OOP), Multi-Agent Systems, Workflow Automation

## 🚀 How to Run the Program
1.  Make sure you have Python installed on your computer.
2.  Download or clone this repository.
3.  Open your terminal or command prompt.
4.  Run the script using:
    ```bash
    python main.py
    ```
5.  The system will automatically simulate the process step-by-step.
